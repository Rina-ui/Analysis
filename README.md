Data Analysis — Kaggle Datasets

Exploration et analyse de datasets publics issus de [Kaggle](https://www.kaggle.com).  
Chaque dossier correspond à un dataset indépendant avec son propre notebook et ses visualisations.


Stack

- **Python** — langage principal
- **Pandas** — manipulation et nettoyage des données
- **Matplotlib / Seaborn** — visualisations
- **Jupyter Notebook** — exploration interactive



> Datasets téléchargés depuis [kaggle.com/datasets](https://www.kaggle.com/datasets) — non inclus dans ce repo.
